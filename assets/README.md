# Assets

Place your favicon and logo here as **`2.png`** (recommended: 32×32 or 192×192 px for favicon; same or larger for logo).

The site references:
- **Favicon / Apple touch icon:** `assets/2.png`
- **Logo (header):** `assets/2.png`

If the file is missing, the browser will show no favicon; add `2.png` to this folder for it to display.
